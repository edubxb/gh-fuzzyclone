# gh fuzzyclone

A [gh](https://github.com/cli/cli) extension to clone GitHub repositories using [fzf](https://github.com/junegunn/fzf#readme).

## Requirements
- [fzf](https://github.com/junegunn/fzf#readme)

## Usage
`gh fuzzyclone`

## Installation
`gh extension install hashue/gh-fuzzyclone`
